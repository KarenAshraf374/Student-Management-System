Student Management System
A robust, console-based desktop application designed to streamline educational administration by managing student records, course offerings, and faculty assignments. Built with a focus on data integrity and modular software design, the system provides real-time search, multi-criteria filtering, and secure entity tracking.

Backend & Logic: C# (Console Application) implementing clean object-oriented design and a modular architecture.

Database & ORM: SQL Server managed via Entity Framework Core, utilizing a code-first approach to handle complex data models and dynamic relationships.

Data Manipulation: High-performance queries and advanced data filtering implemented using LINQ to ensure seamless retrieval and real-time response.

Key Features
Comprehensive CRUD Operations: Full lifecycle management for Students, Courses, Teachers, and Managers.

Enrollment & Relationship Mapping: Seamless execution of student course enrollments and staff course assignments.

Advanced Search & Filtering: Dynamic, multi-tiered search and filtering logic powered by LINQ expressions.
